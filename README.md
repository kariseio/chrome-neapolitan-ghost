# 괴담

나폴리탄 괴담식 크롬 확장. 체험용이고 점프스케어는 없다.

설치하는 순간 열어둔 탭 하나에 "귀신"이 깃들고, 규칙서가 열린다.
귀신은 탭 사이를 옮겨다니고, 규칙을 어길수록 화면이 조금씩 고장난다.

## 설치 (개발자 모드)
1. 주소창 → `chrome://extensions`
2. 우상단 **개발자 모드** ON
3. **압축해제된 확장 프로그램을 로드** → 이 폴더 선택
4. 규칙서가 뜬다. 웹 탭 여러 개를 열어두고 관찰.

권한을 바꾼 뒤에는 확장 카드의 **새로고침(⟳)** 을 눌러야 반영된다.

## 끄기 / 초기화
- **끄기**: `chrome://extensions`에서 토글 OFF 또는 삭제 → 영향받던 탭은 F5로 원복
- **오염도만 리셋**: 규칙서 우하단의 희미한 `·` 버튼

## 구성
- `manifest.json` — 설정 (Manifest V3)
- `background.js` — 귀신 위치·상태 관리, 규칙 감지
- `content.js` — 페이지 연출
- `rules.html` / `rules.js` — 규칙서
