# 탭을 옮겨다니는 것 (PoC)

나폴리탄 괴담식 크롬 확장 프로토타입. 점프스케어 없음.
**설치하는 순간** 열어둔 탭 중 하나에 "귀신"이 깃들고, 동시에 **규칙서**가 열린다.
규칙을 어길수록 그것은 **더 고장난다.**

## 흐름
1. 설치 → 규칙서 탭이 뜨고, http/https 탭 하나의 favicon이 눈으로 바뀐다.
2. 귀신은 한 번에 한 탭(초반)에만 깃들고, 수십 초마다 다른 탭으로 옮겨간다.
3. 규칙을 어기면 **오염도(corruption 0→4)**가 오른다. 오를수록:
   - 이동이 빨라지고 → 여러 탭에 동시에 깃들고
   - 제목이 글리치되고 → 화면 가장자리가 어두워지며 깜빡이고
   - 본문 단어가 더 자주 다른 말로 바뀐다.
   - 규칙서 문서 자체도 훼손되고, **없던 5번째 규칙**이 나타난다.

## 규칙 (어기면 오염도 +1)
| # | 규칙 | 감지 방식 |
|---|------|-----------|
| 1 | 눈이 그려진 탭을 직접 닫지 말 것 | `tabs.onRemoved` |
| 2 | 규칙서 탭을 닫지 말 것 | `tabs.onRemoved` |
| 3 | 탭을 7개 이상 열지 말 것 | `tabs.onCreated` |
| 4 | 눈 탭을 30초 이상 응시하지 말 것 | content script 가시성 타이머 |

오염도는 `storage.local`에 저장되어 **브라우저를 껐다 켜도 유지된다**(고장 누적).

## 설치 (개발자 모드)
1. 주소창 → `chrome://extensions`
2. 우상단 **개발자 모드** ON
3. **압축해제된 확장 프로그램을 로드** → 이 폴더(`D:\study\chrome-ghost`) 선택
4. 규칙서가 뜬다. http/https 탭 여러 개 열고 관찰.

## 테스트 팁
- **느리면**: `background.js`의 `params()` 지연값 또는 오염도를 올려서 확인.
- **오염 빠르게 체험**: 규칙을 일부러 어겨 보라(탭 8개 열기 등). 규칙서가 실시간으로 변한다.
- **초기화**: 규칙서 페이지 우하단의 희미한 `·` 버튼(개발용) → 오염도 0으로 리셋.
- **규칙서 다시 열기**: `chrome://extensions`에서 확장 ID 확인 후 `chrome-extension://<ID>/rules.html`.

## 끄기
`chrome://extensions`에서 토글 OFF 또는 삭제. content script가 흔적(favicon·오버레이)을 복원한다.
오염도까지 지우려면 규칙서의 리셋 버튼을 쓰거나 확장을 삭제하라.

## 파일
- `manifest.json` — 권한/구성 (Manifest V3)
- `background.js` — 서비스 워커. 귀신 위치·오염도 관리, 규칙 위반 감지.
- `content.js` — 페이지 주입. 강도(intensity)별 연출.
- `rules.html` / `rules.js` — 규칙서. 오염도에 따라 훼손된다.

## 알려진 한계 (PoC)
- SW는 상주하지 않음 → `chrome.alarms`로 깨어나므로 이동 간격이 정확히 일정하진 않다.
- favicon이 없는 사이트는 귀신이 떠난 뒤 아이콘이 잠시 비어 보일 수 있다.
- 속삭임/오버레이는 일부 프레임워크(React 등) 페이지에서 즉시 덮어써질 수 있다(해는 없음).
- 규칙 위반 판정은 "사용자가 닫았다"고 가정한다(확장은 스스로 탭을 닫지 않음).
